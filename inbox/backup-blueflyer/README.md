# blueFlyer

An accessible media poster for Bluesky Social with automatic alt text generation for images and GIFs.

## Features

- Automatic alt text generation for images using a small, energy-efficient language model
- GIF support with accessibility features
- Clean, modern interface
- PWA support for installation on any device
- Privacy-focused: No data storage, no training data collection

## Development

This project uses vanilla JavaScript for the frontend with a focus on accessibility and clean code. The styling is done with CSS variables for easy theming and maintenance.

### Setup

1. Clone the repository
2. No build process required - it's vanilla JS and CSS
3. Serve the files using any web server
4. Access the application through your browser

### Structure

- `/css` - Stylesheet files
- `/js` - JavaScript modules
- `/assets` - Images and icons
- `/dev` - Development and testing files

## Accessibility

This project prioritizes accessibility with:
- Semantic HTML
- ARIA labels
- Keyboard navigation
- Screen reader support
- High contrast options
- Responsive design
